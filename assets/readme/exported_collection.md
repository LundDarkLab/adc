# Lund Collection Export

This ZIP archive contains the exported data from your Lund collection.

## Contents

- **collection.csv**  
  All items from the active collection in CSV format (data only).

- **collection.json**  
  The complete collection in JSON format, including metadata and items. You can re-import this file into Lund or share it with others.

- **LICENSE.txt**  
  Usage license: Creative Commons Attribution 4.0 International (CC BY 4.0).

- **README.md**  
  This instruction file.

## How to use

- To **re-import** your collection into Lund, use the \`collection.json\` file with the import function.
- The \`collection.csv\` file can be opened with Excel, LibreOffice, or other spreadsheet software.
- The license file specifies the terms of use for the exported data.
